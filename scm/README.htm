PHP Automatic Photo Albums

The PHP Photo Album Generator is a collection of a few simple interrelated PHP scripts that automate the generation of photo album listings, slideshows and thumbnail index. This includes showing album categories based on which user logged-in. Then, when an album is selected, a slideshow starts with a bunch of viewing options (change speed, show EXIF details, show thumbnail index and movies).

Distinguishing features: automatic generation of albums and categories based on what's installed (no additional configuration). Usage of EXIF comments in the file to show photo descriptions (no need for a database such as MySQL).

Dependencies: EXIF enabled in PHP (modify slideshow.php to remove this dependency), Apache authentication (modify albumlist.php to remove this dependency) and slides/thumbnails installed as subdirectories to album directories (use GenThumbs above for that purpose).

Here's the list of script and their function:

·      index.htm is the starting page that shows all available album categories and provides a starting point. This file shall be installed in a web directory at the same place as all the other PHP scripts. It includes a set of 2 frames with blank.html and albumlist.php in them.

·      albumlist.php shows the list of categories of albums. It actually shows the list of all subdirectories that you want to show based on which user has logged in (using basic htaccess Apache authentication). You therefore need to edit this file to determine which user sees what category(ies).

·      showalbums.php shows the available albums in a second level of directory (album) for a selected subdirectory (category).

·      ranshow.php is there to provide a slideshow of all pictures of the selected category while waiting for the user to select an album.

·      album.php shows the thumbnail index for the album. It also shows movie clips available in the directory in order to allow streaming them also.

·      slideshow.php shows the slide-formatted files of the album an automatic cycles through them.

·      exif.php can be invoked from slideshow.php. It shows EXIF information on the file.

 

Although I haven't spent the time to clean-up these scripts to include all necessary comments, they are usable as is, as long as you follow these instructions:

  1. Install all the files in the package in a specific starting directory
  2. Create a first level of subdirectories for each album category.
  3. Edit albumlist.php to associate album categories to your authorized web users.
  4. Create a second level of subdirectories for each album.
  5. For each album directory, include the original hi-res pictures there and create a slides/ and thumbs/ subdirectory with slide-size and thumbnail-size pictures. Use the thumbnail generator GenThumbs to do this semi-automatically.
